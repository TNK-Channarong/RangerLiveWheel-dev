📢 วิธีตั้งค่า Line Group แจ้งทีม Admin

ขั้นตอนที่ 1 — สร้าง Line Group ใหม่

เปิด Line → กด + → สร้างกลุ่ม → เพิ่ม Line OA (Bot) เข้ากลุ่มด้วย


ขั้นตอนที่ 2 — หา Group ID

ใน Apps Script → เพิ่มฟังก์ชันนี้แล้วรัน:

function getGroupId() {
  // พิมพ์ในกลุ่ม Line แล้วดู log
  Logger.log('Set webhook แล้วส่ง msg ในกลุ่ม');
}

หรือใช้วิธีง่ายกว่า: ตั้ง Webhook ใน Line Developers → ส่งข้อความในกลุ่ม → ดู source ใน Webhook log จะเห็น groupId


ขั้นตอนที่ 3 — ใส่ Group ID ใน Code.gs

const LINE_GROUP_ID = 'C1234567890abcdef...';

ขั้นตอนที่ 4 — Deploy ใหม่ → ทดสอบ spin ดูการแจ้งเตือนในกลุ่ม
